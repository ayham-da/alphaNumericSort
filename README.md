alphaNumericCollator is a compareFunction to sort an Array alphanumeric.

Example : 

    const arr = [" per1son1   /"," per10son10   /"," per22son11   /"," per14son12   /"," per12son13   /"," per12son14   /"," per22son15   /"," per1son2   /"];
    
    alphaNumericCollator(arr)
